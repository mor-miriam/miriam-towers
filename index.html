תמונות מהמחשב נדחסות אוטומטית לגודל אינטרנט.
כשקישור הסליקה ריק, הכפתור מציג הודעת דמו. שליחת המיילים דרך FormSubmit — בפעם הראשונה יישלח אליכם מייל אימות חד־פעמי שיש לאשר.
איך מפרסמים שינויים לכל הגולשים:
1. לחצו «הורדת קובץ מעודכן» — יירד קובץ index.html עם כל התוכן, התמונות והמחירים העדכניים.
2. היכנסו לריפו ב-GitHub → Add file → Upload files.
3. גררו את הקובץ ולחצו Commit changes. תוך דקה האתר מתעדכן.
